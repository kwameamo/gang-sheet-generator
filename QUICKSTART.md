## ğŸ”§ Pro Tips

### Tip 1: Use the Right Unit
- **mm** - European printing, precision work
- **cm** - Quick measurements
- **in** - US printing, standard sizes
- **px** - Digital design, exact pixel control

### Tip 2: Spacing Matters
- **0mm** - Tightly packed (for die-cutting)
- **3-5mm** - Comfortable cutting room
- **10mm+** - Extra safe, easier cutting

### Tip 3: Paper Size Selection
- **A4** - Home printer, testing
- **A3** - Small production runs
- **A2** - Large format, maximum efficiency

### Tip 4: Export Format Choice
- **PDF** - Best for printing (vector-like quality)
- **JPEG** - Smaller file size, universal
- **PNG** - Keeps transparency, best quality

### Tip 5: Save Your Work
- Click "Sign In" (optional Google account)
- After creating gang sheet, click "Save Sheet"
- Access later from "Saved Sheets" tab
- Great for repeat orders!
